## Instrucciones de Instalación
1. **Android:** Crea un proyecto en Android Studio con el paquete `com.cebrameta.soberania`. Copia `NucleoSoberano.kt`.
2. **Interfaz:** En `activity_main.xml`, usa un `Guideline` al 61.8% para separar la pantalla según PHI.
3. **Python:** Ejecuta `matriz_phi_data.py` para inicializar tu base de datos de visiones.
4. **Activación:** Usa el comando de voz para colapsar la función de onda antes de programar.
